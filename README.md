## ✅ Tasks Completed

### Level 1 – Basic
- **Task 1: Data Cleaning & Preprocessing** — cleaned a 497K-row stock prices dataset (missing values, standardized formats).
- **Task 2: Exploratory Data Analysis (EDA)** — summary stats, distributions, correlations on the Boston Housing dataset.

### Level 2 – Intermediate
- **Task 1: Regression Analysis** — simple & multiple linear regression predicting house prices (scikit-learn).
- **Task 3: Clustering Analysis (K-Means)** — unsupervised clustering on the Iris dataset with elbow-method tuning.

### Level 3 – Advanced
- **Task 1: Predictive Modeling (Classification)** — churn prediction using Logistic Regression, Decision Tree, and Random Forest, with GridSearchCV tuning.
- **Task 3: NLP Sentiment Analysis** — text preprocessing + TextBlob polarity-based sentiment classification with word cloud visualization.
